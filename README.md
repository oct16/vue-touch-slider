# vue-touch-slider

> Touch slider plugin for Vue.js

## Install

      npm install vue-touch-slider
      Vue.use(VueTouchSlider)
      
## Usage
#### Available through npm as `vue-touch-slider`.
  
      import vueTouchSlider from 'vue-touch-slider'  
      Vue.use(VueTouchSlider)


#### Using the `v-touch-slider` directive

    <div v-touch-slider>
    	content...
    	<img>...	
    </a>
